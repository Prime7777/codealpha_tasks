## 🗂️ Tasks Overview

### 🌸 Task 1 — Iris Flower Classification
[![Task1](https://img.shields.io/badge/Status-Completed-brightgreen?style=flat-square)](./Task1_IrisFlowerClassification)

> Classify Iris flowers into 3 species using Machine Learning

| Item | Detail |
|------|--------|
| 📁 Dataset | Kaggle — UCI Iris Dataset (150 samples) |
| 🤖 Model | Support Vector Machine (SVM) |
| 🎯 Accuracy | **94.74%** |
| 📚 Libraries | Scikit-learn, Pandas, Matplotlib |

---

### 📊 Task 2 — Unemployment Analysis
[![Task2](https://img.shields.io/badge/Status-Completed-brightgreen?style=flat-square)](./Task2_UnemploymentAnalysis)

> Analyze unemployment trends in India & investigate Covid-19 impact

| Item | Detail |
|------|--------|
| 📁 Dataset | Kaggle — Unemployment in India (768 records) |
| 📈 Analysis | Time series, Regional, Rural vs Urban |
| 🦠 Covid Spike | **+3.52%** increase after lockdown |
| 📚 Libraries | Pandas, Matplotlib, NumPy |

---

### 🚗 Task 3 — Car Price Prediction
[![Task3](https://img.shields.io/badge/Status-Completed-brightgreen?style=flat-square)](./Task3_CarPricePrediction)

> Predict used car selling prices using Regression models

| Item | Detail |
|------|--------|
| 📁 Dataset | Kaggle — Used Cars Dataset (301 cars) |
| 🤖 Best Model | Random Forest Regressor |
| 🎯 R² Score | **~95%** |
| 📚 Libraries | Scikit-learn, Pandas, Matplotlib |
